# URL Generator

A simple URL Generator powered by modern technologies.

### Run the app

 #### `docker-compose up -d`
 #### `docker-compose logs localtunnel`

## Technologies

 ### Frontend

  - `Bulma`

 ### Backend

  - `Redis`
  - `Fastify`

 ### Putting it all together

  - `Docker`
  - `docker-compose`
  - `localtunnel`
